# Arpan Christian – Portfolio Website

A personal portfolio website built with pure HTML, CSS and JavaScript.

## Sections

- **Hero** – Name, animated typed role titles, social links and CTA buttons
- **About** – Bio, key details and animated stat counters
- **Skills** – Categorised skill tags (Frontend, Backend, Database, Tools)
- **Projects** – Six project cards with icons, descriptions, tech tags and links
- **Contact** – Contact info panel and a validated contact form
- **Footer** – Social links and dynamic copyright year

## Tech Stack

| Layer      | Technology          |
|------------|---------------------|
| Markup     | HTML5               |
| Styling    | CSS3 (custom properties, CSS Grid, Flexbox) |
| Scripting  | Vanilla JavaScript  |
| Icons      | Font Awesome 6      |
| Fonts      | Google Fonts – Inter |

## Running Locally

No build step required — simply open `index.html` in any modern browser:

```bash
# optional: serve with a local dev server
npx serve .
```
